# Tic-Tac-Toe Game

Visit the [Live Demo](https://maqboolwebdev-stack.github.io/Tic-Tac-Toe/)

This Tic-Tac-Toe project is part of **The Odin Project**.

It was one of the most challenging projects I have worked on so far. Before this, I had never built an application with this level of architecture, so I spent many days designing, debugging, and improving it.

In this project, I practiced several new JavaScript concepts, including **IIFEs**, **encapsulation**, and **separating game logic from the UI**. This project helped me better understand how to organize code into independent modules and write more maintainable JavaScript.
